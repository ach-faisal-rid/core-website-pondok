# Sprint 2 — Site Management

Fokus: semua identitas website dapat diubah tanpa mengubah kode.

## Site Setting

Halaman `/admin/settings` dengan tab:

* [x] **Umum** — nama, tagline, logo, favicon, footer, SEO
* [x] **Beranda** — hero (judul, subtitle, tombol, background), statistik dinamis
* [x] **Profil** — hero, visi, misi (list), motto, nilai, panca jiwa (tambah/hapus), pengasuh
* [x] **Kontak** — telepon, WA, email, alamat, maps, sosial

### Statistik

Admin dapat menambah/menghapus statistik (nilai, judul, ikon).

### Panca Jiwa

Setiap item: judul, deskripsi, ikon — bisa tambah/hapus.

### Visi / Misi / Motto / Nilai Pondok

Semuanya editable di tab Profil.

> Setelah sprint ini, identitas website tidak lagi tergantung hardcode di kode.
