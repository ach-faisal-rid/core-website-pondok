# Contoh pengisian menu Website

Contoh di bawah ini sama dengan data seed. Setelah `php artisan db:seed`, field teks sudah terisi seperti ini. Operator tinggal ganti nama/angka sesuai pondok asli, lalu unggah gambar.

---

## 1. Hero

Tampil di paling atas beranda.

| Field | Contoh isi |
| --- | --- |
| Judul | `Mencetak Generasi Beradab, Berilmu, dan Berwibawa di Era Digital.` |
| Subtitle | `Membangun karakter santri melalui pendidikan Islam yang kokoh, adab yang luhur, dan kesiapan menghadapi tantangan zaman.` |
| Background | Unggah foto gedung/kegiatan pondok (JPG/PNG lebar) |
| Tombol 1 - teks | `Jelajahi Profil` |
| Tombol 1 - URL | `/profil` |
| Tombol 2 - teks | `Hubungi Kami` |
| Tombol 2 - URL | `/kontak` |

**Tips URL:** pakai path lokal (`/profil`), bukan link panjang, kecuali ke situs luar.

## Alur untuk anak / operator

1. Login admin → sidebar **Website**
2. Buka menu satu per satu (mulai Hero)
3. Lihat field yang sudah terisi seed
4. Ganti teks ke data pondok asli
5. Unggah gambar yang masih kosong
6. Klik **Simpan**
7. Buka **Lihat Website** untuk cek hasil
